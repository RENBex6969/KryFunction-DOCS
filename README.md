* Kry.getHWID()

```lua

```

* Kry.getIP()
Returns the client IP
```lua
print(Kry.getIP())
```

* Kry.getExecutorName()
Returns the client executor name.
```lua
print(Kry.getExecutorName())
```

* Kry.getExecutorLVL()

